Adjusting for Publication Bias with JASP

The following analyses are based on Appendix A: Example Report from Bartoš et al. (2022), who reanalyzed a meta-analysis of acculturation mismatch and intergenerational cultural conflict (Lui, 2015). Note that this file was updated in JASP version 0.95 leading to minor differences from the published manuscript.


Lui (2015) studied how acculturation mismatch (i.e., the contrast between the collectivist cultures of Asian and Latin immigrant groups and the individualistic culture of the United States) correlates with intergenerational cultural conflict by meta-analyzing 18 independent studies correlating acculturation mismatch with intergenerational cultural conflict. A standard random-effects reanalysis calculated with a restricted maximum likelihood estimator and using Fisher z values transformed from correlation coefficients indicates a significant relationship between acculturation mismatch on increased intergenerational cultural conflict, ρ = 0.250, 95% CI [0.172, 0.336], p < .001. The correlation coefficient was calculated by back-transforming the Fisher z summary effect size estimate of 0.254 [95% CI (0.170, 0.324)]. According to Cochran's Q-test for residual heterogeneity, the true outcomes appear to be heterogeneous (Q(17) = 73.58, p < .001, τ2 = 0.02, I2 = 77.8%). Lui (2015) visually inspected a funnel plot and concluded, "The funnel plot [...] revealed a symmetrical ``funnel'' shape, supporting that sample size bias [i.e., small study effects] was unlikely for this meta-analysis''. Furthermore, Lui (2015) conducted a moderator analysis using publication status and found no evidence for a systematic difference in means between published and unpublished studies.

PET-PEESE

First, we analyze the data using PET-PEESE to adjust for the relationship between effect sizes and standard errors, correcting for the small study effects (Stanley & Doucouliagos, 2014; Stanley, 2017).

We proceed with the analysis in the following way; we test for the presence of the effect with PET (see the Test of Effect table), if the PET's test for the presence of the effect is not statistically significant at α = 0.10, we interpret the PET's effect size estimate, if the PET's test for the presence of the effect is statistically significant, we interpret the PEESE's effect size estimate.

Selection Models

Second, we analyze the data using selection models to adjust for publication bias operating on p-values (Vevea & Hedges, 1995). We will use significance level α = 0.10 for the test of publication bias (Renkewitz & Keiner, 2019) and α = 0.05 for the tests of heterogeneity and effect sizes. We estimated the two-sided selection models with p-value cut-offs set to 0.05 and 0.10 and automatically joined p-value intervals.

We proceed with the analysis in the following way; we test for the presence of heterogeneity (see the Test of Heterogeneity table) and follow-up with the test of publication bias (see the Test of Publication Bias table), if the test of heterogeneity is statistically significant, we interpret the test assuming heterogeneity, if the test if heterogeneity is not statistically significant, we interpret the test assuming homogeneity. Finally, we interpret the effect size estimate based on the selected combination of the absence/presence of heterogeneity and publication bias.

Robust Bayesian Meta-Analysis

Third, we analyze the data with robust Bayesian meta-analysis (Maier et al., 2023; Bartoš et al., 2023), specifically the RoBMA-PSMA model ensemble that allows us to combine the PET-PEESE and selection models publication bias adjustment with Bayesian model-averaging.

We set the Effect size measure option to Fisher's z and specify the Effect Size and Effect Size Standard Error variables. The remaining settings, i.e., the Bayesian Model-Averaging options and Prior Distributions options are kept at the defaults which corresponds to the RoBMA-PSMA model ensemble.

Model Summary

The Model Summary output contains summary of the model ensemble. The prior model probabilities, posterior model probabilities, and inclusion Bayes factors for the individual model components (i.e., pooled effect, heterogeneity, and publication bias) are summarized in the Meta-Analytic Tests table. The the model-averaged meta-analytic estimates, i.e., the pooled effect size and between-study heterogeneity estimate are summarized Meta-Analytic Estimates table. Additional estimates, such as the publication bias adjustment model parameters such as the relative publication probabilities for different p-value intervals of selection models can be obtained by selecting the Weight function estimates and PET-PEESE estimates in the Statistics section.

We find the absence of evidence for the presence of the effect, BF10 = 1.32, extreme evidence for heterogeneity, BFrf = 30002, and moderate evidence for publication bias, BFpb = 5.00. Under the assumption of equal prior model probabilities for the absence vs. presence of each component, P(M), our posterior probability about the presence of the effect does not increase much, P(Effect|data) = 0.569, while our posterior probability of the between-study heterogeneity, P(Heterogeneity|data) = 1.000, becomes almost certain, and our posterior probability about the presence of publication bias increases notably to P(Publication bias|data) = 0.833.

The resulting model-averaged effect size estimate is ρ = 0.102, 95% CI [0.000, 0.292], with the heterogeneity estimate, τ = 0.161, 95% CI [0.081, 0.292]. Note that we already transformed the pooled effect estimate to correlation scale by selection the "Fisher's z to r" Transform effect size option in the Statistics section.


These estimates are model-averaged across models, assuming both the presence and absence of the effect. In other words, if models assuming the absence of the effect retain a substantial posterior model probability, P(M|Data) in the Model Summary table, the estimates and confidence intervals are shrunk towards the null hypothesis of no effect (ρ = 0). Model-averaged posterior effect size estimate assuming the presence of the effect and the posterior between-study heterogeneity estimate assuming the presence of heterogeneity can be obtained by selecting the Conditional estimates option. 


Prior and Posterior Plots

The Prior and Posteri Plots section provides visual summaries of the posterior distributions. Selecting the Effect size option creates the model-averaged prior and posterior distribution of the pooled effect size. The height of the arrows corresponds to the prior (grey) and posterior (black) probability (right y-axis) of the null hypothesis that puts point distribution at ρ = 0. The continuous densities correspond to the prior and posterior distributions (density, left y-axis) under the alternative hypothesis.

Diagnostics

The MCMC Diagnostics section provides diagnostic summaries for the MCMC estimation procedure. Selecting the Overview table checkbox creates the Diagnostics Summary table. We find that all MCMC diagnostics are good, with all R-hat values below 1.01 and all ESS above 500, where the commonly used rule of thumb for a good R-hat is < 1.05 and ESS > 500 (Gelman, Rubin, et al., 1992; McElreath, 2020).

In conclusion, robust Bayesian meta-analysis extends the findings of PET-PEESE and selection models by finding absense of evidence for the presence of the effect and by providing a model-averaged effect size estimate incorporating the uncertainty about the best publication bias adjustment model.
