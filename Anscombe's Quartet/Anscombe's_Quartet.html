Anscombe's Quartet: Importance of Visual Inspection of Data

Description:

This data set, "Anscome's Quartet", provides the classic example of the need to visualize one's data.

This JASP file highlights the need to inspect data thoroughly using plots rather than statistical output. The data were presented by Anscombe (1973) and to this day the complete data set is known as "Anscombe's Quartet".

The data consist of four variations of two variables (x and y). Across the four variations, the variables x have the same means and standard deviations, which is true also for the variables y. Furthermore, the four correlations and regression equations are also the same. Despite these surface similarities, the data patterns are very different, somethin that becomes apparent as soon as one inspects the scatter plots.

References:

Anscombe, F. J. (1973). Graphs in statistical analysis. The American Statistician, 27: 17–21.

Wagenmakers, E. J. (2015). A quartet of interactions. Cortex, 73, 334-335.
