BCG Vaccine: Fixed-Effects and Random-Effects Meta-analysis

Description:

This data set, "BCG Vaccine", provides test statistics from studies examining the effectiveness of a BCG vaccine.

Variables:

  • trial - The study number.
  • author - Authors of the study.
  • year - Year of publication.
  • tpos - The number of TBC tests that came out positive in the treatment (vaccine) condition. 
  • tneg - The number of TBC tests that came out negative in the treatment (vaccine) condition. 
  • cpos - The number of TBC tests that came out positive in the control (no vaccine) condition. 
  • cneg - The number of TBC tests that came out negative in the control (no vaccine) condition.
  • ablat - Geographical lattitude of the study.
  • alloc - Patient allocation (random, alternate, or systematic).
  • ES - Effect size (log-odds ratios) computed from the trial outcomes.
  • SE - Standard error of the effect size.

This example JASP file demonstrates the use of fixed-effects and random-effects meta-analysis. For information on fitting more complex models, visit the original blog post by Raoul Grasman (https://jasp-stats.org/2017/11/15/meta-analysis-jasp/).

Classical Meta-Analysis

The fixed-effects meta-analysis produces two tables. The first table provides the omnibus tests for the estimated coefficients. The second table provides the meta-analytic estimates of the coefficients.

Fixed and Random Effects
  Q df p
Omnibus test of Model Coefficients 106.536 1 < .001
Test of Residual Heterogeneity 163.194 12 < .001
Note.  p -values are approximate.
Note.  The model was estimated using Fixed Effects method.

The first row in the first table tests the null hypothesis that all coefficients in the second table are equal to zero. In this case, we have only an intercept, therefore it is just a test of the null hypothesis that the mean effect size is zero. Because the test is statistically significant, we reject the null hypothesis.

The second row in the first table checks the heterogeneity of the effect sizes. Statistically significant result means that it is highly unlikely to observe such heterogeneity (or more extreme), if the variance of the observed effect sizes was produced solely by sampling error. This indicates that the fixed-effects model is not suitable for the data.

Note.  Wald test.

Given that the fixed-effects model is not suitable for the data, we cannot generalize the estimate. However, the fixed-effect estimate can be still interpreted to mean that the average effect size of the observed studies is not zero.
