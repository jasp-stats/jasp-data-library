Plot Builder: Boxplot with p values

This example shows how to use the Plot builder module to create boxplots with p values.

The data set contains 150 observations of three iris flower types: Setosa, Versicolor, and Virginica. Four features were measured from each flower in the data: the length and the width of the sepals and petals, in centimeters. 

  1. Sepal.Length: The length of the flower's sepal in cm.
  2. Sepal.Width: The width of the flower's sepal in cm.
  3. Petal.Length: The length of the flower's petal in cm.
  4. Petal.Width: The width of the flower's petalin cm.

Because we know that there are 3 different flower types in the data, we build a fixed 3-cluster model on the data set.


Suppose that we have previously performed an ANOVA test and Tukey's post hoc tests to obtain p values.


Plot Builder (beta)

  1. Add individual data points → This option is available under “Individual data points.”
  2. Add a boxplot → You can enable this under “Distributions.”
  3. Color by X variable → Set this under “Theme and color.”
  4. Hide the legend, as it is redundant → You can disable it under “Legend.”
  5. Add p-values → Go to “P value brackets.” Be sure to click the green plus button and enter the exact names of the groups you wish to compare.
  6. Note: If p-values added on the second or third attempt appear cut off or extend beyond the plot, you’ll need to increase the axis padding. This setting is located under “Size, margins and plot orientation.”
