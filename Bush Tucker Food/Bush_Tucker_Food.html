Bush Tucker Food: One-way Repeated Measures ANOVA

Description:

This fictional data set, "Bush Tucker Food", provides the numer of seconds before celebrities retch while eating disgusting food - each celebrity consumed four differen kinds of disgusting items.

Variables:

  • Celebrity - Identification number of a celebrity.
  • Stick Insect - The number of seconds before a celebrity retched while eating stick insect.
  • Kangaroo Testicle -The number of seconds before a celebrity retched while eating kangaroo testicle.
  • Fish Eye - The number of seconds before a celebrity retched while eating fish eye.
  • Witchetty Grub - The number of second before a celebrity retched while eating witchetty grub.

This example JASP file demonstrates the use of repeated measures ANOVA. Specifically, we asses the adequacy of the null hypothesis that the four kinds of food are equally repelling to celebrities.

Reference:

Field, A. P. (2017). Discovering Statistics Using IBM SPSS Statistics (5th ed.). London: Sage.

Repeated Measures ANOVA

jaspAnova::AnovaRepeatedMeasures(
        version = "0.18",
        contrasts = list(list(contrast = "repeated", variable = "Animal")),
        descriptivePlotErrorBar = TRUE,
        descriptivePlotHorizontalAxis = "Animal",
        descriptivePlotYAxisLabel = "Seconds to Retch ",
        descriptives = TRUE,
        effectSizeEstimates = TRUE,
        effectSizeEtaSquared = FALSE,
        plotHeightDescriptivesPlotNoLegend = 386,
        plotWidthDescriptivesPlotLegend = 450,
        plotWidthDescriptivesPlotNoLegend = 571,
        postHocCorrectionBonferroni = TRUE,
        postHocCorrectionHolm = FALSE,
        postHocPooledError = FALSE,
        postHocTerms = list("Animal"),
        repeatedMeasuresCells = list("Stick Insect", "Kangaroo Testicle", "Fish Eye", "Witchetty Grub"),
        repeatedMeasuresFactors = list(list(levels = list("Stick Insect", "Kangaroo Testicle", "Fish Eye", "Witchetty Grub"), name = "Animal")),
        sphericityCorrectionGreenhouseGeisser = TRUE,
        sphericityCorrectionHuynhFeldt = TRUE,
        sphericityTests = TRUE,
        withinModelTerms = list("Animal"))

A repeated-measures ANOVA with Greenhouse-Geisser correction (see Assumption checks) indicates that the observed differences between the kinds of food are not statistically significant at the .05 level, F = 3.794, p = 0.063.
