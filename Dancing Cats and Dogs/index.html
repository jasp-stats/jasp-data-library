Dancing Cats and Dogs: Log-linear Analysis

Description:

This fictional data set, "Dancing Cats and Dogs", provides the number of cats and dogs who learned to dance, trained by one of two types of conditioning.

Variables:

  • Animal - Species of animal trained (Cat, Dog).
  • Training - Type of conditioning (Food as reward, Affection as reward).
  • Dance - Did the animal learn to dance? (Yes, No).

This example JASP file demonstrates the use of a log-linear analysis. We will examine which type of training is better for cats and dogs using a log-linear analysis.

Reference:

Field, A. P. (2017). Discovering Statistics Using IBM SPSS Statistics (5th ed.). London: Sage.

The ANOVA table shows the tests of whether including a specific term into the analysis leads to a statistically significant improvement. We can see that the model with the three-way interaction is statistically significant. This means that the saturated model outperforms the other models and we need to investigate the three-way interaction, which suggests that the effect of training on learning to dance is different for cats and dogs.
