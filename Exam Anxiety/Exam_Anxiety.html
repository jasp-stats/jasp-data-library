Exam Anxiety: Pearson's Correlation

Description:

This fictional data set, "Exam Anxiety", provides questionnaire scores by students prior to an exam (the variables are anxiety, preparedness, and grade).

Variables:

  1. Code - Participant's identification number.
  2. Revise - Time spent studying for the exam (in hours).
  3. Exam - Performance in the exam (percentages).
  4. Anxiety - Anxiety prior to the exam as measured by the Exam Anxiety Questionnaire.
  5. Gender - Participant's gender (Male, Female).

This example JASP file demonstrates the use of the Pearson correlation coefficient. Specifically, we will investigate how the three measured variables correlate with each other.

Field, A. P. (2017). Discovering Statistics Using IBM SPSS Statistics (5th ed.). London: Sage. [Fictional data set]

Correlation

The results show a statistically significant negative correlation between Exam and Anxiety (r = -0.441, p < 0.01): the lower the Anxiety, the higher the Exam score.
