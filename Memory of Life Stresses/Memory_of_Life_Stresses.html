Memory of Life Stresses: Multinomial Test


Multinomial Test

H₀ (a) 45.367 17 < .001

Note.  Chi-squared approximation may be incorrect

The null hypothesis [here denoted as H0(a)] that Habermans (1978) tested entails that the probability that participants report a stressful life event is equal for all 18 months prior to the interview. We can reject this null hypothesis with chi-square(17) = 45.367, p < .001.

The second chi-square goodness-of-fit test [denoted as H0(b)] indicates that we can reject the null hypothesis that the proportions of reported life events are in line with our expected proportions, with chi-square(17) = 84.673, p < .001. Note that we can either manually enter our expected values or drag the variable that encodes our expectations into the field 'Expected counts'. The entered values can be expected counts, expected proportions or any number that reflects our expectations. JASP automatically normalizes the entered values.
