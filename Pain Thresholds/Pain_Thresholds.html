Pain Thresholds: One-way ANOVA

Description:

This data set, "Pain thresholds", provides pain threshold for people with different hair color. 

Variables:

  1. V1 - Participant number.
  2. Hair Color - Hair color of the participant (`Light Blond' = light blond, `Dark Blond' = dark blond, `Light Brunette' = light brunette, `Dark Brunette' = dark brunette).
  3. Pain Tolerance - Performance on pain sensitivity test (the higher the score, the better the pain tolerance).

This example JASP file demonstrates the use of one-way ANOVA. We will conduct both classical version and a Bayesian version of the analysis.

Reference:

McClave, J. T. and Dietrich, F. H. (1991). Statistics. San Francisco: Dellen publishing.
