Plot Builder: Plot layout

This example demonstrates how to create a clear and well-structured plot layout using the Plot Builder, based on penguin data collected by scientists between 2007 and 2009 from three islands in the Palmer Archipelago in Antarctica.


  1. island [categorical] - which of the islands (Biscoe, Dream, Torgersen) the data was collected from.
  2. bill_length_mm - length of the 'culmen' (dorsal ridge on the top of the bill)
  3. bill_depth_mm - depth of the culmen
  4. flipper_length_mm [continuous]
  5. body_mass_g [continuous] - rounded to the nearest 5g
  6. sex [categorical] - labels 'male' and 'female'
  7. year [ordinal] - 2007, 2008, 2009


Horst AM, Hill AP, Gorman KB (2020). palmerpenguins: Palmer Archipelago (Antarctica) penguin data. R package version 0.1.0. https://allisonhorst.github.io/palmerpenguins/

Plot Builder (beta)

This guide walks you through the process of creating a multi-panel layout using Plot Builder, based on individual plots such as Plot 1 and Plot 2. These plots are automatically assigned names (e.g., Plot 1, Plot 2, etc.) that are used later when arranging them into a combined layout.


1. Create Individual Plots

Plot 1 settings:

  1. Set species on the x-axis and bill_length_mm on the y-axis.
  2. Under “Individual data points”, check:
  3. Individual data points
  4. Adjust point size, jitter height/width, and transparency as needed.
  5. Under “Distributions”, check:
  6. Boxplot
  7. Customize settings such as line width, box width, whisker width, and outlier display.

Plot 2 settings:

  1. Set bill_length_mm on the x-axis and bill_depth_mm on the y-axis.
  2. Under “Individual data points”, check:
  3. Individual data points
  4. Add ellipse (e.g., type = Normal, level = 0.95, transparency = 0.2)
  5. Use Color by variable (e.g., species) under “Theme and color” to distinguish groups.

Important: These individual plots will be saved as Plot 1, Plot 2, etc. You will need these exact names when creating the combined layout.


2. Combine Plots into a Layout

  1. Open the “Plot layout” panel.

To arrange plots in a row:

  1. In the “Arrange plots by row” section, click the green + button.
  2. Under “Row 1”:
  3. Enter the plot names in the Plot IDs field (e.g., Plot 1, Plot 2)
  4. Add labels such as A, B in the Labels field (optional).
  5. You can also set Plot widths if you want to adjust their relative size.

Optional settings:

  1. Change Label size, Horizontal position, and Vertical position to control subplot label placement.
  2. Set overall Width, Height, and Spacing for the final figure.
  3. Check “Collect legend” if you want to merge the legends from the individual plots.
