Recidivism and Mental Health in Delinquent Juveniles

Description:

This dataset includes 17 studies (with multiple effect sizes per study) comparing recidivism rates among delinquent juveniles with vs. without a mental health disorder, across different types of delinquent behavior. Since multiple effect size estimates were extracted from most studies (e.g., for different delinquent behaviors in which juveniles could have recidivated), we need to account for the dependency structure. In this example JASP file, we analyze the Assink's data using a multilevel multivariate meta-analysis with cluster robust standard error to account for the dependency and examine whether the effect size differs between the types of the delinquent behavior.


Variables:

study: numeric identifier for each study

esid: within-study effect size identifier

id: unique row identifier

yi: standardized mean difference in recidivism between juveniles with and without a mental health disorder (positive values indicate more recidivism in the mental health disorder group).

sei: standard error of the effect size estimate.

pubstatus: whether the study was published (yes/no)

year: publication year (centered around the mean)

deltype: type of delinquent behavior (general, overt, or covert)


References:

Assink, M., van der Put, C. E., Hoeve, M., de Vries, S. L. A., Stams, G. J. J. M., & Oort, F. J. (2015). Risk factors for persistent delinquent behavior among juveniles: A meta-analytic review. Clinical Psychology Review, 42, 47–61. https://doi.org/10.1016/j.cpr.2015.08.002


Viechtbauer W, White T, Noble D, Senior A, Hamilton W (2025). metadat: Meta-Analysis Datasets. doi:10.32614/CRAN.package.metadat, R package version 1.4-0, https://CRAN.R-project.org/package=metadat.

Classical Meta-Analysis (Multilevel/Multivariate)

To account for the dependency between the extracted effect size estimates, we specify both the hierarchical random effects structure accounting for the nesting of effect sizes within studies and the working variance-covariance matrix accounting for the correlation between within and between the different delinquent behavior scales applied to the same participants.


To specify the working variance-covariance matrix, we use the Effect Size Variance-Covariance Matrix Section and select the Constructs, groups, and times option. This menu allows us to specify a wide range of dependencies between effect size estimates (fully described in the help file and the documentation of the metafor R package). Here, we require only the Construct variable that separates between different measures of the same construct and Construct Type variable that separates between different measures of the same construct type, both within the same Cluster. To finish the specification, we need to enter the assumed (known) within construct and between construct correlation using the Construct Correlation Matrix and Construct Type Correlation Matrix options.


To specify the hierarchical random effects structure, we use the Random Effects / Model Components Section. This menu allows us to specify the Nested (multilevel) dependency and select the nesting of effect sizes (esid, Level 2) within studies (study, Level 1).


Finally, the cluster-robust standard errors are obtained by setting the study variable in the Clustering option in the data input section.


We can examine the resulting metafor call by selecting the Show metafor R code option in the Advanced section.

Forest Plot

The Forest Plot section allows us to visualize the effect size estimates from individual studies alongside the estimated marginal means for each type of delinquent behavior and the overall model summary. We select the Aggregate by Study option to create a more compact forest plot and visualize the individual effect size estimates as Bubbles. As a result, we obtain a single row with multiple effect sizes (bubbles) per study, greatly reducing the length of the forest plot. 

			
			effectSizeVarianceCovarianceMatrix <- vcalc(

				



				vi = samplingVariance,

				



				data = dataset,

				



				cluster = study,

				



				obs = esid,

				



				rho = list(0.7, 0.5),

				



				type = deltype

				



			)

				



				



			fit <- rma.mv(

				



				yi = yi,

				



				V = effectSizeVarianceCovarianceMatrix,

				



				data = dataset,

				



				mods = ~deltype,

				



				random = list(`Component 1` = ~1 | study/esid),

				



				method = 'REML',

				



				test = 't',

				



				level = 95

				



			)

				



				



			fit <- robust(

				



				fit,

				



				cluster = study,

				



				clubSandwich = TRUE,

				



				adjust = TRUE

				



			)

In summary, we conducted a multilevel multivariate meta-analysis with cluster-robust standard errors to assess the recidivism rates among delinquent juveniles with vs. without a mental health disorder, across different types of delinquent behavior. We demonstrated how to specify the working variance-covariance matrix and how to specify the hierarchical random effects. Finally, we displayed the results in a compact forest plot aggregating the effect size estimates within studies.
