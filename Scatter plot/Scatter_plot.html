Plot Builder: Scatter plot with p values

This example shows how to use the Plot builder module to create a scatter plot with p value.

Description:

This data set, "Big Five Personality Traits", provides scores of 500 participants on a Big Five personality questionnaire.

Variables:

  1. Neuroticism - Score on the Neuroticism scale.
  2. Extraversion - Score on the Extraversion scale.
  3. Openness - Score on the Openness to Experience scale.
  4. Agreeableness - Score on the Agreeableness scale.
  5. Conscientiousness - Score on the Conscientiousness scale.


We performed a Pearson correlation analysis and found a negative association between Neuroticism and Conscientiousness (r = –0.368, p < .001).


Dolan, C. V., Oort, F. J. Stoel, R. D, and Wicherts, J. M, (2009) Testing measurement invariance in the target rotated multigroup exploratory factor model, Structural Equation Modeling: A Multidisciplinary Journal, 16, 295-314.  

Plot Builder (beta)

  1. Add data points → Enable this under “Individual data points.”
  2. Add curve fit → This can be found under “Curve fit and reference line.”
  3. Note: If the scatterplot is very dense, the fitted curve (in the same color as the points) might not be clearly visible. In such cases, it is recommended to use the “Black lines” option for better contrast.
  4. Add the p-value and correlation coefficient → Go to “Annotation and data label.”
  5. Don’t forget to click on the green plus icon to insert the values. Important: For scatter plots, the “P value brackets” option cannot be used, as it requires a categorical X-axis variable.
