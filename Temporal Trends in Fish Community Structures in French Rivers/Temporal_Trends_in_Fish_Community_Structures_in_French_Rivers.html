Temporal Trends in Fish Community Structures in French Rivers

Description

This data set, "Studies on Temporal Trends in Fish Community Structures in French Rivers," contains results from studies examining changes in the abundance of fish species over time in French rivers. The data include measurements from 35 sampling stations across 11 sites along various rivers, with observations spanning 19 to 37 years up to 2015. The primary focus is on understanding how fish communities have shifted over time, potentially due to factors like climate change.


In this example JASP file, we perform a meta-analysis that accounts for both nested random effects (sampling stations within sites) and spatial correlation among the sampling stations. This approach allows us to properly model the dependencies in the data due to the hierarchical structure and spatial relationships between sampling locations.


Variables

site: Character identifier for each study site.

station: Character identifier for each sampling station at a site.

site_station: Combined identifier for site and station.

s1: Mann-Kendall trend statistic for the relative abundance of non-local species.

vars1: Corresponding sampling variance for s1, corrected for temporal autocorrelation.

s2: Mann-Kendall trend statistic for the relative abundance of northern species.

vars2: Corresponding sampling variance for s2, corrected for temporal autocorrelation.

s3: Mann-Kendall trend statistic for the relative abundance of non-native species.

vars3: Corresponding sampling variance for s3, corrected for temporal autocorrelation.


We add the se variable corresponding to the standard errors of the effect size estimates using the "Add Column" functionality in the JASP data editor (using the square root of the vars1 variable).

Classical Meta-Analysis (Multilevel/Multivariate)

To analyze these data, we set up a meta-analytic model that accounts for both nested random effects and spatial correlation among the sampling stations.


Nested Random Effects: We include random effects for sites and stations nested within sites. This accounts for the hierarchical structure of the data, recognizing that measurements from stations within the same site may be more similar to each other than those from different sites. We specify the nested random effects by adding the Nested (multilevel) Type Structure in the Random Effects / Model Structure Section.


Spatial Random Effects: We model spatial correlation among the sampling stations using a spatial covariance structure. The spatial correlation is based on the distances between stations, with closer stations expected to have more similar trends than those further apart. We specify the nested random effects by adding the Spatial - Gaussian Type Structure in the Random Effects / Model Structure Section.


Loading the Distance Matrix

The spatial correlation is specified using a distance matrix that provides the distances between all pairs of sampling stations. In this case, we load the distance matrix from a CSV file. The CSV file needs to be set-up in the following way: the first and the first column has to contain names corresponding to the Location Identifiers provided in the data set. This allows JASP to match the distance matrix entries with the corresponding observations. Here is an example of the couple first rows and columns of the data set:

The Advanced section contains additional options that allow us to inspect the generated metafor R code via the Show metafor R code option and also extend the metafor R code with additional commands. Here, we extend the metafor R code to facilitate model convergence and improve estimation by providing an initial value for the spatial correlation parameter. This is done by adding the code snippet control = list(rho.init=10) to the model setup. This initial value helps the optimization algorithm find the best-fitting parameters more efficiently. In general, the code snippet must be a named list of options that is appended to the function call (here, the enclosing list call is ommited, the full call would be list(control = list(rho.init=10)) - multiple options must be separated by a comma within the list).

Our multilevel meta-analysis accounting for spatial correlation reveals that fish communities in French rivers are experiencing significant changes over time. The positive overall trends in the abundance of certain fish species suggest ecological shifts that could be a response to ongoing global changes.
