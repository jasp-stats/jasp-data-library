Decision Tree Classification

This example demonstrates decision tree classification of penguins into species, using data collected by scientists between 2007 and 2009 from three islands in the Palmer Archipelago in Antarctica. We have the following predictor variables (features):


  • island [categorical] - which of the islands (Biscoe, Dream, Torgersen) the data was collected from.
  • bill_length_mm - length of the 'culmen' (dorsal ridge on the top of the bill)
  • bill_depth_mm - depth of the culmen
  • flipper_length_mm [continuous]
  • body_mass_g [continuous] - rounded to the nearest 5g
  • sex [categorical] - labels 'male' and 'female'
  • year [ordinal] - 2007, 2008, 2009

 The decision tree outputs a predicted species for each row in the data, based on the values of the preictor variables. The mechanism for this is described below.

Horst AM, Hill AP, Gorman KB (2020). palmerpenguins: Palmer Archipelago (Antarctica) penguin data. R package version 0.1.0. https://allisonhorst.github.io/palmerpenguins/
