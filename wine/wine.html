Random Forest Clustering

The wine dataset contains the results of a chemical analysis of wines grown in a specific area of Italy. Three types of wine are represented in the 178 samples, with the results of 13 chemical analyses recorded for each sample. The recorded variables are:

  1. Alcohol: The amount of alcohol in the wine
  2. Malic: Malic acid.
  3. Ash: Ash.
  4. Alcalinity: Alcalinity of ash.
  5. Magnesium: Magnesium.
  6. Phenols: Total phenols.
  7. Flavanoids: Flavanoids.
  8. Nonflavanoids: Nonflavanoid phenols.
  9. Proanthocyanins: Proanthocyanins.
  10. Color: Color intensity.
  11. Hue: Hue.
  12. Dilution: D280/OD315 of diluted wines.
  13. Proline: Proline.

This example shows a Random forerst clustering algorithm to identify the most distinguishing variables between wines. More specifically, the random forest cluster model is optimized with respect to its BIC value, which can be inspected in the Elbow Curve Plot.

Elbow Method Plot

The elbow plot tells us where to find the kink in the curve, the point where considering another cluster would be unneccesary. The red dot represents the minimum BIC value, since the model is optimized with respect to this metric.

t-SNE Cluster Plot

A t-SNE plot is way to visualize highly dimensional data on a two dimensional graph. The price to pay, however, is that the axes become unterpretable. This t-SNE plot nicely shows how the different clusters are mapped together.
